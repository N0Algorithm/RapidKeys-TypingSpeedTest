# ⚡ RapidKeys

### Turn your typing into a superpower. 

RapidKeys is more than just a typing test. It's a clean, minimal, and lightning-fast space designed to help you master your keyboard. Built with Next.js and a focus on performance, RapidKeys gives you the precision data you need to level up your speed without the clutter.

---

## ✨ Features that feel right
- **Instant Flow**: No loading screens, no distractions. Just start typing.
- **Deep Insights**: Beyond just WPM. Dive into your accuracy, streaks, and identify your weak spots with our error heatmap.
- **The Replay Room**: Watch your performance back in real-time. See exactly where you hesitated and where you excelled.
- **Tailored for You**: Adjustable test lengths and settings that let you practice your way.
- **Client-Side Magic**: Everything runs right in your browser. Fast, private, and efficient.

---

## 🚀 Getting Started

Want to run RapidKeys locally? It's easy.

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) (version 18 or higher) installed.

### Setup
1. **Clone and Install**
   ```bash
   npm ci
   ```

2. **Launch the Engine**
   ```bash
   npm run dev
   ```

3. **Start Typing**
   Head over to [http://localhost:3000](http://localhost:3000) and show us what you've got.

## 🛠️ Built with Precision
- **Framework**: Next.js (App Router)
